# AngularPrimeNgBase

This is a base project for Angular 18 with PrimeNg 12.1.1. to start a new project.

## Current Version

* Angular 18
* PrimeNg 17.18.8
* PrimeIcons 7.0.0
* TailwindCSS 3.4.9

## Theming

To change the theme, you can change the `angular.json` file and change the `styles` array in the `build` and `test` objects.

```json
"styles": [
    "src/styles.css",
    "node_modules/primeicons/primeicons.css",
    "node_modules/primeng/resources/themes/saga-blue/theme.css",
    "node_modules/primeng/resources/primeng.min.css",
],
```
